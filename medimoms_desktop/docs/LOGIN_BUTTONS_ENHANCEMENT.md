## 🎨 Color Psychology

### Emerald Green (#10B981)
- **Healthcare**: Associated with health, growth, healing
- **Trust**: Calming and trustworthy
- **Action**: Encourages positive action
- **Professional**: Modern and clean

### Button Contrast
- **Primary (Solid)**: High contrast, draws attention
- **Secondary (Outlined)**: Clear but less prominent
- **Hierarchy**: Clear visual priority

## 🎓 Design Principles Applied

### 1. Visual Hierarchy
- Primary action (Sign In) is solid and prominent
- Secondary action (Register) is outlined and clear
- Users naturally focus on Sign In first

### 2. Consistency
- Both buttons use emerald green theme
- Same height and border radius
- Consistent spacing
- Unified design language

### 3. Affordance
- Icons suggest button purpose
- Clear visual feedback on hover
- Obvious clickable elements
- Intuitive interaction

### 4. Accessibility
- High contrast text
- Large click targets
- Clear visual states
- Keyboard navigation support
